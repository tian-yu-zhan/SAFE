This is the README file for the R code to replicate results in the article “Synergy Area with FDR-controlled Evaluation (SAFE) to robustly assess safety profile in clinical trials”

Source the R code in the folder “1_sim_1” to replicate results in Section 4 of the main article.
Data for Section 5 of the main article are obtained via TransCelerate. After obtaining data, authors will further share the code of this section with corresponding recipients.
Source the R code in the folder “2_sim_2_power” to replicate results in Section 1 of Supplementary Materials.
Source the R code in the folder “3_sim_3_cor” to replicate results in Section 2 of Supplementary Materials.
Source the R code in the folder “4_sim_4_alt” to replicate results in Section 3 of Supplementary Materials.
